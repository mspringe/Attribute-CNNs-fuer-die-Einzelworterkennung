src.estimation.base module¶

In this module we introduce a base class for estimation, that is solely used for duck typing, as well as a base class for an easy implementation of Metric based estimators.


class src.estimation.base.DistEstimator(words, metric, phoc_level=3, alphabet=[<Alphabet.ASCII_LOWER: 1>, <Alphabet.ASCII_DIGITS: 4>, <Alphabet.ASCII_PUNCTUATION: 3>], unambiguous=False)[source]¶

Bases: src.estimation.base.Estimator

A DistEstimator shall perform a nn search, with respect to a metric. Distance estimators are of almost identical behaviour. Their only hyper-parameter is the metric used.
