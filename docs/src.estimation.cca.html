src.estimation.cca module¶

This module provides a class to carry out the regularised CCA, lexicon based approach.


class src.estimation.cca.RCCAEstimator(words, n_dim=128, reg=10000.0, metric=<Metrices.COSINE: 3>, phoc_lvls=3, alphabet=[<Alphabet.ASCII_LOWER: 1>, <Alphabet.ASCII_DIGITS: 4>, <Alphabet.ASCII_PUNCTUATION: 3>])[source]¶

Bases: src.estimation.base.Estimator

The RCCAEstimator performs a regularized CCA and a nearest neighbour search on the transformed data

Note

The RCCAEstimator additionally logs its configuration in a json file, when saved

estimate_set(X, normalize=True)[source]¶

Estimation of an entire Set. This should work better, than estimating samples individually, due to disparity in the normalization of the attributes. The dataset X and the PHOC of the lexicon will be transformed into the subspace in batches of the same size, to have simmilarly normalization behaviour.

Parameters

  • X – Queries, to be estimated

  • normalize – Indicates whether to apply normalization after estimation

Returns

List of estimated words
