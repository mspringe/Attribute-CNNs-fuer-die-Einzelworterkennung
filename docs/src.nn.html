Module contents¶

This package provides modules for the PHOCNet, Pyramidal Pooling and a STN-PHOCNet model.

The pp module provides a class for Pyramidal Pooling and different approaches thereof. The phocnet module provides an implementation of the PHOCNet in the torch framework and the stn an implementation of a STN, that can be used a s an initial Spatial Transformer layer in the PHOCNet.
